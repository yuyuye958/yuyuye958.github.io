git入门

###几个常见命令的使用方法

  1. ls 罗列
    ls 路径
    ls -a 路径
    ls -l 路径
    ls -al 路径
  2. cat
    使用“cat 文件名”命令查看文件内容
  3. mv 移动 / 重命名
    用法:mv 源路径 目标路径
    mv 1.txt 2.txt 这样我们就把 1.txt 移到 2.txt 了，也就是重命名

  4. touch 触摸
    创建文件 touch 文件名
    改变文件更新时间 touch 文件名
    touch 1.txt 创建文件
